# OK Lab Nürnberg

Das **OK Lab Nürnberg** ist ein Teil des Netzwerks [Code for Germany](https://codefor.de/). "OK" steht für Open Knowledge. Man kennt uns auch als **Open Data Nürnberg**, **Code for Nürnberg** oder abgekürzt **C4N**.
